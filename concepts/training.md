# <a name="microsoft-graph-training"></a>Aprendizaje de Microsoft Graph

Los ejercicios de aprendizaje de Microsoft Graph son tutoriales paso a paso que le ayudarán a crear una aplicación básica que usa la autenticación de Azure Active Directory y accede a los datos a través de Microsoft Graph.

Los siguientes tutoriales de aprendizaje están disponibles actualmente.

- [ASP.NET](https://docs.microsoft.com/graph/training/aspnet-tutorial/)
- [Microsoft Flow](https://docs.microsoft.com/graph/training/flow-tutorial/)
- [Node.js](https://docs.microsoft.com/graph/training/node-tutorial/)
- [PHP](https://docs.microsoft.com/graph/training/php-tutorial/)
- [Python](https://docs.microsoft.com/graph/training/python-tutorial/)
- [Ruby](https://docs.microsoft.com/graph/training/ruby-tutorial/)
- [React](https://docs.microsoft.com/graph/training/react-tutorial/)
- [UWP](https://docs.microsoft.com/graph/training/uwp-tutorial/)

¿No ve su idioma preferido ni su plataforma? Solicite un tutorial para ellos en [UserVoice](https://officespdev.uservoice.com/forums/224641-feature-requests-and-feedback/category/101632-microsoft-graph-o365-rest-apis).
