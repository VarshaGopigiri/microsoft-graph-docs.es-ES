# <a name="accessaction-resource-type"></a>tipo de recurso accessAction

> **Importante:** Las API de la versión /beta de Microsoft Graph son una versión preliminar y están sujetas a cambios. No se admite el uso de estas API en aplicaciones de producción.

La presencia del recurso en un [**itemActivity**] **accessAction** [ activity] indica que la actividad de acceso a un elemento.

>**Nota:** Registros de actividad de Access actualmente sólo están disponibles en SharePoint y OneDrive para la empresa.

[activity]: itemactivity.md

## <a name="properties"></a>Propiedades

Este tipo de recurso no tiene propiedades.
