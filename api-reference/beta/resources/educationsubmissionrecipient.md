# <a name="educationsubmissionrecipient-resource-type"></a>tipo de recurso educationSubmissionRecipient

> **Importante:** Las API de la versión /beta de Microsoft Graph son una versión preliminar y están sujetas a cambios. No se admite el uso de estas API en aplicaciones de producción.

Clase abstracta que representa los distintos conjuntos de usuarios al que se puede asignar un envío. Los valores posibles incluyen individuos o grupos. Esto se usa dentro de la propiedad **submission.recipient** .


## <a name="properties"></a>Propiedades
Ninguno.
