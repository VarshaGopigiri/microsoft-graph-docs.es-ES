#  <a name="certificationcontrol-resource-type"></a>tipo de recurso certificationControl

Cumplimiento de normas de contiene datos de certificación asociados a garantizar el control de puntuación.

|Propiedad |Tipo |Descripción |
|:--|:--|:--|
|name | string | Nombre del control de certificación |
|url | string | Portal de confianza de la dirección URL del servicio de Microsoft |
