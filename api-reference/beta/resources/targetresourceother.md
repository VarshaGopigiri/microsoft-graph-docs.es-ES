# <a name="targetresourceother-resource-type"></a>tipo de recurso targetResourceOther
Indica información en un destino que no se dividen en el resto de las categorías de recursos de destino como usuario, aplicación, rol, grupo, clave. Deriva el recurso [targetResource](targetresource.md) .



## <a name="properties"></a>Propiedades
