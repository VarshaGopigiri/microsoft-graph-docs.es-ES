# <a name="educationassignmentrecipient-resource-type"></a>tipo de recurso educationAssignmentRecipient

> **Importante:** Las API de la versión /beta de Microsoft Graph son una versión preliminar y están sujetas a cambios. No se admite el uso de estas API en aplicaciones de producción.

Se usa dentro de la propiedad **assignments.assignTo** . Esta superclase es abstracta y no se puede establecer la propiedad en las asignaciones a este valor. Esta superclase representa quién se debe asignar la asignación cuando se publicó la asignación.


El recurso [educationAssignmentClassRecipient](educationassignmentclassrecipient.md) es una subclase de esta superclase.

## <a name="properties"></a>Propiedades
Ninguno.
