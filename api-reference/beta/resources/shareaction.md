# <a name="shareaction-resource-type"></a>Tipo de recurso ShareAction

> **Importante:** Las API de la versión /beta de Microsoft Graph son una versión preliminar y están sujetas a cambios. No se admite el uso de estas API en aplicaciones de producción.

El recurso **ShareAction** proporciona información sobre un recurso [activity][activity] que compartió un elemento.

[activity]: itemactivity.md

## <a name="json-representation"></a>Representación JSON

<!-- {
  "blockType": "resource",
  "optionalProperties": [ ],
  "@type": "microsoft.graph.shareAction"
}-->

```json
{
  "recipients": [{"@odata.type": "microsoft.graph.identitySet"}]
}
```

## <a name="properties"></a>Propiedades

| Nombre de la propiedad | Tipo                       | Descripción
|:--------------|:---------------------------|:-----------------------------
| recipients    | Colección [identitySet][] | Las identidades con las que se compartió el elemento en esta acción.

[identitySet]: identityset.md

## <a name="remarks"></a>Comentarios

Actualmente, los registros de actividad de elementos solo están disponibles en SharePoint y OneDrive para la Empresa.
