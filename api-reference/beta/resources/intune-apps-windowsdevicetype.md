Contiene las propiedades de tipo de dispositivo de Windows.

|Miembro	|Valor|Descripción|
|:---|:---|:---|
|ninguno|0|Sin marcadores establecidos.|

|equipo|8|Si no se admite el tipo de dispositivo de Windows del equipo.|
