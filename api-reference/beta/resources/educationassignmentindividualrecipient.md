# <a name="educationassignmentindividualrecipient-resource-type"></a>tipo de recurso educationAssignmentIndividualRecipient

> **Importante:** Las API de la versión /beta de Microsoft Graph son una versión preliminar y están sujetas a cambios. No se admite el uso de estas API en aplicaciones de producción.

Se usa dentro de la propiedad [assignment.assignTo](educationassignment.md) . Cuando se establece en lista de destinatarios individual, estudiantes seleccionados en la clase recibirán un objeto de envío cuando se publicó la asignación.

Este recurso es una subclase de [educationAssignmentRecipient](educationassignmentrecipient.md).

## <a name="properties"></a>Propiedades

Ninguno.
