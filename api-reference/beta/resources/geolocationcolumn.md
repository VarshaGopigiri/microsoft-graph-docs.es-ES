# <a name="geolocationcolumn-resource-type"></a>tipo de recurso geolocationColumn

> **Importante:** Las API de la versión /beta de Microsoft Graph son una versión preliminar y están sujetas a cambios. No se admite el uso de estas API en aplicaciones de producción.

El **geolocationColumn** en un recurso [columnDefinition](columndefinition.md) indica que la columna contiene una ubicación geográfica.

}

## <a name="properties"></a>Propiedades

El recurso **geolocationColumn** no contiene ninguna propiedad.
