# <a name="sharedpcaccountdeletionpolicytype-enum-type"></a>Tipo de enumeración sharedPCAccountDeletionPolicyType

> **Nota:** El uso de las API de Microsoft Graph para configurar las directivas y los controles de Intune requiere que el cliente tenga la [licencia correcta](https://go.microsoft.com/fwlink/?linkid=839381) para el servicio Intune.

Valores posibles para cuando se eliminan cuentas en un PC compartido.
## <a name="members"></a>Miembros
|Miembro|Valor|Descripción|
|:---|:---|:---|
|immediate|0|Eliminar inmediatamente.|
|diskSpaceThreshold|1|Eliminar en el umbral de espacio en disco.|
|diskSpaceThresholdOrInactiveThreshold|2|Eliminar en el umbral de espacio en disco o umbral inactivo.|
