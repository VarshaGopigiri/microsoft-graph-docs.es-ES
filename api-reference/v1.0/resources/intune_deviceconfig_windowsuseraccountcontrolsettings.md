# <a name="windowsuseraccountcontrolsettings-enum-type"></a>Tipo de enumeración windowsUserAccountControlSettings

> **Nota:** El uso de las API de Microsoft Graph para configurar las directivas y los controles de Intune requiere que el cliente tenga la [licencia correcta](https://go.microsoft.com/fwlink/?linkid=839381) para el servicio Intune.

Configuración de control de la cuenta de usuario de Windows.
## <a name="members"></a>Miembros
|Miembro|Valor|Descripción|
|:---|:---|:---|
|userDefined|0|Definido por el usuario, valor predeterminado, sin intención.|
|alwaysNotify|1|Notificar siempre.|
|notifyOnAppChanges|2|Notificar cuando haya cambios en la aplicación.|
|notifyOnAppChangesWithoutDimming|3|Notificar cuando haya cambios en la aplicación y sin atenuar el escritorio.|
|neverNotify|4|No notificar nunca.|
