# <a name="applisttype-enum-type"></a>Tipo de enumeración appListType

> **Nota:** El uso de las API de Microsoft Graph para configurar las directivas y los controles de Intune requiere que el cliente tenga la [licencia correcta](https://go.microsoft.com/fwlink/?linkid=839381) para el servicio Intune.

Posibles valores de la lista de aplicaciones de cumplimiento.
## <a name="members"></a>Miembros
|Miembro|Valor|Descripción|
|:---|:---|:---|
|none|0|Valor predeterminado, sin intención.|
|appsInListCompliant|1|La lista representa las aplicaciones que se considerarán compatibles (solo las aplicaciones de la lista son compatibles).|
|appsNotInListCompliant|2|La lista representa las aplicaciones que se considerarán no compatibles (todas las aplicaciones son compatibles excepto las aplicaciones de la lista).|
