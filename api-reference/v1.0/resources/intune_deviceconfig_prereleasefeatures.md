# <a name="prereleasefeatures-enum-type"></a>Tipo de enumeración prereleaseFeatures

> **Nota:** El uso de las API de Microsoft Graph para configurar las directivas y los controles de Intune requiere que el cliente tenga la [licencia correcta](https://go.microsoft.com/fwlink/?linkid=839381) para el servicio Intune.

Valores posibles para las características de versión previa.
## <a name="members"></a>Miembros
|Miembro|Valor|Descripción|
|:---|:---|:---|
|userDefined|0|Definido por el usuario, valor predeterminado, sin intención.|
|settingsOnly|1|Configuraciones de características preliminares solo.|
|settingsAndExperimentations|2|Configuraciones y experimentos de características preliminares.|
|notAllowed|3|Características preliminares no permitidas.|
