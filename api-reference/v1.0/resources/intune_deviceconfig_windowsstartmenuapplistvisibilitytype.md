# <a name="windowsstartmenuapplistvisibilitytype-enum-type"></a>Tipo de enumeración windowsStartMenuAppListVisibilityType

Tipo de visibilidad de lista de aplicación de menú de inicio.
## <a name="members"></a>Miembros
|Miembro|Valor|Descripción|
|:---|:---|:---|
|userDefined|0|Definido por el usuario. Valor predeterminado.|
|collapse|1|Contraer la lista de aplicaciones en el menú Inicio.|
|remove|2|Quita la lista de aplicaciones en su totalidad del menú Inicio.|
|disableSettingsApp|4|Deshabilita la alternancia correspondiente (contraer o quitar) en la aplicación Configuración.|
