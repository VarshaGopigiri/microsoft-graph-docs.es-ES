Etiquetas de clasificación de contenido de TV en Irlanda
## <a name="members"></a>Miembros
|Miembro	|Valor|Descripción|
|:---|:---|:---|
|allAllowed|0|Valor predeterminado, permitir que todos los TV muestra contenido|
|allBlocked|1|No permitir que cualquier TV muestra contenido|
|general|2|La clasificación de GA es adecuada para todas las audiencias|
|secundario|3|La clasificación de canales es adecuada para los elementos secundarios|
|youngAdults|4|La clasificación YA es adecuada para adolescentes|
|parentalSupervision|5|La clasificación de PS invita a los padres y tutores a tener en cuenta el acceso de restricción hijos|
|¿implementación consolidada|6|La clasificación MA es adecuada para adultos|
