Etiquetas de clasificación de contenido de TV en Irlanda
## <a name="members"></a>Miembros
|Miembro|Valor|Descripción|
|:---|:---|:---|
|allAllowed|0|Valor predeterminado, permitir todo el contenido de los programas de TV|
|allBlocked|1|No permitir ningún contenido de programas de TV|
|general|2|La clasificación GA es apta para todos los públicos|
|children|3|La clasificación CH es apta para niños|
|youngAdults|4|La clasificación YA es apta para el público adolescente|
|parentalSupervision|5|La clasificación PS invita a los padres y tutores a tener en cuenta la restricción de acceso para niños|
|mature|6|La clasificación MA es apta para adultos|
