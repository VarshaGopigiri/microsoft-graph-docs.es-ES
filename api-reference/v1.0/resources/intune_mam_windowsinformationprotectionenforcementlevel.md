# <a name="windowsinformationprotectionenforcementlevel-enum-type"></a>Tipo de enumeración windowsInformationProtectionEnforcementLevel

> **Nota:** El uso de las API de Microsoft Graph para configurar las directivas y los controles de Intune requiere que el cliente tenga la [licencia correcta](https://go.microsoft.com/fwlink/?linkid=839381) para el servicio Intune.

Valores posibles para los niveles de cumplimiento de WIP
## <a name="members"></a>Miembros
|Miembro|Valor|Descripción|
|:---|:---|:---|
|noProtection|0|Sin cumplimiento de protección|
|encryptAndAuditOnly|1|Solo cifrado y auditoría|
|encryptAuditAndPrompt|2|Cifrado, auditoría y solicitud de confirmación|
|encryptAuditAndBlock|3|Cifrado, auditoría y bloqueo|
