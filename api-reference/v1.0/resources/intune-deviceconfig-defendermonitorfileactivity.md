Valores posibles para supervisar la actividad de archivo.
## <a name="members"></a>Miembros

|:---|:---|:---|

|deshabilitar|1|Deshabilitar supervisar la actividad de archivo.|
|monitorAllFiles|2|Supervisar todos los archivos.|
|monitorIncomingFilesOnly|3| Supervisar sólo los archivos entrantes.|
|monitorOutgoingFilesOnly|4|Supervisar sólo los archivos de salida.|
