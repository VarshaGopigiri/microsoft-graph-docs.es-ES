# <a name="androidworkprofilecrossprofiledatasharingtype-enum-type"></a>Tipo de enumeración androidWorkProfileCrossProfileDataSharingType

> **Nota:** El uso de las API de Microsoft Graph para configurar las directivas y los controles de Intune requiere que el cliente tenga la [licencia correcta](https://go.microsoft.com/fwlink/?linkid=839381) para el servicio Intune.

Tipo de uso compartido de datos entre perfiles para el perfil de trabajo de Android.
## <a name="members"></a>Miembros
|Miembro|Valor|Descripción|
|:---|:---|:---|
|deviceDefault|0|Valor predeterminado del dispositivo, sin intención.|
|preventAny|1|Impedir cualquier uso compartido.|
|allowPersonalToWork|2|Permite la solicitud de uso compartido de datos desde el perfil personal al perfil de trabajo.|
|noRestrictions|3|No hay restricciones en el uso compartido.|
