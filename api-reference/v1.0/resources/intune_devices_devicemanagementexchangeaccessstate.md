# <a name="devicemanagementexchangeaccessstate-enum-type"></a>tipo de enumeración deviceManagementExchangeAccessState

> **Nota:** El uso de las API de Microsoft Graph para configurar las directivas y los controles de Intune requiere que el cliente tenga la [licencia correcta](https://go.microsoft.com/fwlink/?linkid=839381) para el servicio Intune.

Estado de acceso de dispositivos de Exchange.
## <a name="members"></a>Miembros
|Miembro|Valor|Descripción|
|:---|:---|:---|
|none|0|No hay ningún estado de acceso detectado desde Exchange|
|unknown|1|Se desconoce el estado de acceso del dispositivo a Exchange|
|allowed|2|El dispositivo tiene acceso a Exchange|
|blocked|3|El dispositivo está bloqueado en Exchange|
|quarantined|4|El dispositivo está en cuarentena en Exchange|
