# <a name="managedappclipboardsharinglevel-enum-type"></a>Tipo de enumeración managedAppClipboardSharingLevel

> **Nota:** El uso de las API de Microsoft Graph para configurar las directivas y los controles de Intune requiere que el cliente tenga la [licencia correcta](https://go.microsoft.com/fwlink/?linkid=839381) para el servicio Intune.

Representa el nivel al que el Portapapeles del dispositivo puede compartirse entre aplicaciones
## <a name="members"></a>Miembros
|Miembro|Valor|Descripción|
|:---|:---|:---|
|allApps|0|Se permite el uso compartido entre todas las aplicaciones, administradas o no.|
|managedAppsWithPasteIn|1|Se permite el uso compartido entre todas las aplicaciones administradas con pegar en habilitado|
|managedApps|2|Se permite el uso compartido entre todas las aplicaciones administradas.|
|blocked|3|El uso compartido entre aplicaciones está deshabilitado|
