# <a name="windowsarchitecture-enum-type"></a>Tipo de enumeración windowsArchitecture

> **Nota:** El uso de las API de Microsoft Graph para configurar las directivas y los controles de Intune requiere que el cliente tenga la [licencia correcta](https://go.microsoft.com/fwlink/?linkid=839381) para el servicio Intune.

Contiene las propiedades de la arquitectura de Windows.
## <a name="windowsarchitecture-flags"></a>Marcas de windowsArchitecture
|Miembro|Valor|Descripción|
|:---|:---|:---|
|none|0|Sin marcadores establecidos.|
|x86|1|Si el tipo de arquitectura de Windows X86 es compatible o no.|
|x64|2|Si el tipo de arquitectura de Windows X64 es compatible o no.|
|arm|4|Si el tipo de arquitectura de Windows Arm es compatible o no.|
|neutral|8|Si el tipo de arquitectura de Windows Neutral es compatible o no.|
