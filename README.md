# <a name="microsoft-graph-documentation"></a>Documentación de Microsoft Graph

Gracias por su interés en la documentación de Microsoft Graph. Para que la experiencia sea óptima, le recomendamos que vea este contenido en el [portal para desarrolladores de Microsoft Graph](https://developer.microsoft.com/es-ES/graph/docs/concepts/overview).

## <a name="give-us-your-feedback"></a>Envíenos sus comentarios

Su opinión es importante para nosotros.

- Para informarnos sobre preguntas o problemas que encuentre en la documentación, [envíe un problema](https://github.com/microsoftgraph/microsoft-graph-docs/issues) en este repositorio.
- También le animamos a realizar una bifurcación, corregir los errores y realizar una solicitud de incorporación de cambios. En [CONTRIBUTING.md](CONTRIBUTING.md) encontrará las instrucciones de colaboración.
- Para enviarnos comentarios sobre su experiencia de programación, lo que le gustaría ver en versiones futuras o ejemplos de código, escriba sus sugerencias e ideas en [Office Developer Platform UserVoice](https://officespdev.uservoice.com/forums/224641-feature-requests-and-feedback?category_id=101632).
